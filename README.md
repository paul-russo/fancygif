# fancygif

Tiny shell function to turn a video file into a GIF.

## Dependencies

[FFmpeg](https://github.com/FFmpeg/FFmpeg) and [gifsicle](https://github.com/kohler/gifsicle)

## Installation

Add this to your `~/.zshrc` file:
```
zplug "paul-russo/fancygif"
```

## Usage

```
fancygif video.mov
```

That's it.
